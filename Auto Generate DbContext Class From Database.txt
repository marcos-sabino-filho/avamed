Auto Generate DbContext Class From Database | Scaffold-DBContext | Database First


Script

Criar pasta Database

Instalar com NuGet:
Microsoft.EntityFrameworkCore.SqlServer
Microsoft.EntityFrameworkCore.Tools


No terminal do gerenciador de pacotes do NuGet:
Scaffold-DbContext "ConnectionString" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Database -Context ApplicationDbContext -DataAnnotations
